1. Описание 

Используя развитые возможности form validation и javascript, придумать и 
разработать форму для ввода различных данных о себе и своих предпочтениях. Сделать 
кнопку для отправки данных на сервер. Предусмотреть второй этап валидации уже на 
самом сервере.

2. Инструкция

Проект выполнялся мною для Windows. 
Я сделала проект по вашему первому выложенному заданию, где про ubuntu не было ничего сказано,
зашла недавно вижу - уже под ubuntu. Под ubuntu я не настраивала web-сервер. 

ASP.NET в Ubuntu : http://it-sea.ru/web-development/install-asp.net-linux.htm 

Либо же поставить в Ubuntu виртуальную машину, установить на ней Windows 7, 
а дальше руководствоваться этим :

Поднятие web-сервера на Windows 7:

Панель управления->Программы и компоненты->Включение и выключение компонентов Windows
1) Microsoft Framework 3.5 (поставить галочку)
2) Службы IIS->Службы интернета и Средства управления веб-сайтом (поставить галочки и нажать ОК)

Дальше идем в "Панель управления\Все элементы панели управления\Администрирование"->"Диспетчер служб IIS"
Переходим во вкладку Default Web site, нажимаем слева "начало" (сервер поднимется на 127.0.0.1:80).

Зайти в папку C:\inetpub\wwwroot 
Cкачать отсюда https://github.com/fetchhell/institute в эту папку

теперь сайт доступен по адресу http://localhost:63016/lab1_web.aspx
